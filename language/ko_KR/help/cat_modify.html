사진첩고치기

정보

  • 이름: 사진첩이름고치기 (물리적사진첩의 경우, 가상이름고치기)
  • 설명: 내용의 일반적의견을 알리는 짤막한 글로, 이 글은 마중그림쪽의 아래에 표시됩니다.

옮기기

사진첩이 가상이라면, 옮길수 있습니다. 사진첩옮기기는 상위사진첩을 바꿀수 있다는 뜻입니다.

선택지

  • 다가가기속성: 권한관리. 사진첩을 비공개로 돌리면, 하위사진첩들도 모두 비공개로 됩니다. 사진첩을 공개한다면, 하위사진첩들도 모두 공개로 됩니다.
  • 자물쇠: 사진첩과 하위사진첩은 잠시동안 유지보수를 위해 쓸수없게 됩니다.
  • 덧글: 인가된 사용자들이 이 사진첩의 사진에 덧글달수 있습니다.

설정관리화면에서 이 선택지들을 관리할수 있습니다 (화면 덧글, 자물쇠, 공개여부, 대표 쓸수있는곳: 관리 » 사진첩 » 권한).

정렬순

기본그림정렬순을 씁니다.

이 정렬순이 하위사진첩에도 적용될지 나타냅니다.

이 사진첩과 쓰일 특정정렬기준을 선택합니다.

  • 작성날짜: 사진찍은 날짜
  • 올림날짜: 동기화날짜
  • 평균비률 (*): 평균비률은 들러뵈기수에 따라 바뀔수 있습니다
  • 들러뵘수 (*): 지금의 들러뵘에 따라 가장많은 들러뵘순서가 바뀝니다
  • 파일이름: 정보란에 주어진 이름
  • 식별자: 안쪽식별자 (최근분류가 이전것보다 식별자가 높습니다. )

(*)경고: 이런 정렬순을 시험하는데는 예상치않은 결과가 나올수 있습니다.

Representant

The album representative is the thumbnail displayed on the main page (category.php) to represent the album when it contains only sub-albums and no direct photo (just as the root album).

photo of a album :

  • 사진페지: 활동단추는 one of the action buttons let you set the current photo as the representative for the displayed album. This button is available only for administrators.
  • 사진정보: 관리화면에서 변경합니다. 이 화면은 picture.php 또는 로부터 다다를수 있습니다. Batch management in unit mode. See the help of this screen for details.
  • 관리 » 사진첩 » 환경설정, Representative. 이 화면에의 자세내용은 도움말에서 보세요.
  • Edit album (the present screen).

대표고름은 allow_random_representative 환경설정매개변수에 따라 달라집니다 (see include/config_default.inc.php).

In default mode (allow_random_representative set to false), each album containing at least one element is represented by a fixed element. Once set (at album creation), the representative changes only when an administrator ask for it. If the representative is not appropriate, you can ask to Find a new representant by random.

If the allow_random_representative configuration parameter is set to true, a category containing photos may have no fixed representant. Just use Delete representant button.

If the album contains only sub-albums and no photo, it can nonetheless be represented by any element thanks to Modify informations about a photo screen. The only option on the current screen is the Delete representant button.

모든 사진첩사진을 새 사진첩에 잇습니다

  • 가상사진첩이름: 새 사진첩의 이름이 만들어져, 모든 지금분류의 사진들이 새것으로 이어집니다.
  • 상위사진첩: 새 사진첩이 있을곳은, 맨위에다 만들도록 비워둡니다.

모든 사진첩속사진을 존재하는 사진첩에다 잇습니다

  • 사진첩: 목적사진첩을 고릅니다.
