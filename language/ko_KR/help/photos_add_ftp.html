빠른 시작
  1. 컴퓨터에 디렉토리 만들기.
  2. 이 디렉토리에 일부 사진을 복사하고 웹 표시를 위해 크기 조정. 경고: 디렉토리 및 파일 이름은 문자만 포함 해야 합니다., 숫자, "-", "_"또는 ".". 공백이나 악센트 부호가있는 문자 안됌.
  3. FTP 클라이언트를 사용하여 Piwigo 설치파일의 "갤러리" 디렉터리에 당신의 디렉터리를 복사합니다.
  4. 갤러리에 로그온하고 관리 로 이동합니다 그리고 큰 동기화 버튼을 클릭합니다.

축하합니다! 사진 갤러리의 첫 번째 앨범을 성공적으로 만들었습니다.

디렉터리 및 파일 구성

  • 앨범 디렉토리는 Piwigo의 "갤러리" 디렉토리에 있습니다. 다음은 아주 작은 갤러리의 디렉토리 트리입니다. (그러나 많은 기능 사용):

    galleries
|-- wedding
|   |-- celebration
|   |   |-- arrival
|   |   |   |-- paul-alone.jpg
|   |   |   +-- sarah-alone.jpg
|   |   +-- departure
|   |       +-- paul-and-sarah.jpg
|   +-- party
|       |-- dancers001.jpg
|       |-- dancers002.jpg
|       +-- dancers003.jpg
+-- honeymoon
|   |-- hotel.png
|   |-- video-from-plane.avi
|   +-- pwg_representative
|       +-- video-from-plane.jpg
+-- photoshoot
    |-- img0001.jpg
    |-- img0002.jpg
    +-- pwg_format
        |-- img0001.cr2
        |-- img0001.cmyk.jpg
        |-- img0001.zip
        |-- img0002.cr2
        +-- img0002.cmyk.jpg
  • "pwg_representative"및 "pwg_format"(아래 설명 참조)을 제외하고 아래의 각 "갤러리" 디렉토리는 앨범을 생성합니다. 깊이에 제한이 없습니다.
  • 기본적으로, 요소는 파일로 표시됩니다. A file can be a Piwigo element if its extension is among file_ext configuration parameter possibilities (see include/config_default.inc.php file). A file can be a picture if its extension is among picture_ext configuration parameter possibilities.
  • 사진이 아닌 요소 (비디오, 사운드, 파일 텍스트 등 원하는 항목 ...)는 기본적으로 파일 이름 확장자에 해당하는 아이콘으로 표시됩니다. 선택적으로 썸네일과 대표 파일을 연결할 수 있습니다. (위 예제의 video.avi 참조)
  • 여러 형식 : 사진은 여러 형식으로 제공 될 수 있습니다. 이 예를 보면 img0001.jpgfmf에 대해 3 개의 추가 형식이 있습니다.로컬 구성에서 $ conf [ 'enable_formats'] = true; 를 사용하여이 기능을 활성화하고 $ conf [ 'format_ext'] = array ( 'cmykjpg ','cr2 ','zip '); 와 같은 형식 목록을 정의 할 수 있습니다.
  • 경고: 디렉토리 및 파일 이름에는 문자, 숫자, "-", "_"또는 "."만 포함되어야합니다. 공백이나 악센트 부호가있는 문자는 사용 할 수 없습니다
  • 사진이 디렉토리에 올바르게 배치되면, go to: Administration » Tools » Synchronize
