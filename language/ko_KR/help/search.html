검색

이 페이지에서 전체 갤러리의 사진을 검색 할 수 있습니다.

단어 검색
갤러리 사진과 관련된 속성 중 하나 또는 여러 단어를 검색합니다.부분 일치의 경우 *를 와일드 카드로 사용.
저자 검색
부분 일치의 경우 *를 와일드 카드로 사용.
날짜 검색
쿼리의 날짜 및 / 또는 종료 날짜를 선택하십시오. "이전"쿼리를 작성하려는 경우 날짜 필드를 비워 둡니다. 마지막 필드의 연도는 AAAA 형식으로 입력해야합니다. (i.e. 2004)
앨범에서 검색
검색하려는 앨범을 선택하십시오. 모든 하위 앨범은 상위 앨범을 선택하고 아래 하위 앨범 검색 활성화를 설정하여 검색 할 수 있습니다 .
